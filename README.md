# Negotiate Events Field

A library to negotiate the `Events` header field as proposed in the [Events Query](https://cxres.github.io/events-query-test/draft-gupta-httpapi-events-query.html) Internet-Draft.

## Installation

Install **Negotiate Events Field** using your favourite framework or package manager:

```bash
bun|deno|npm|pnpm|yarn add negotiate-events-field
```

or if you like to live on the bleeding edge:

```sh
bun|npm|pnpm|yarn add github:CxRes/negotiate-events-field
```

In deno, you may import the package directly using the repository URL.

Check out the [docs](./docs/) folder for API documentation and code examples.
